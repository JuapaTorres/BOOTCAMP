# Alke Wallet

Proyecto de billetera digital desarrollado para el bootcamp.
Incluye funcionalidades de depósito, transferencia y consulta de movimientos.

## Tecnologías utilizadas:

- HTML5 (Semántico)
- CSS Personalizado con Google Fonts
- Bootstrap 5.3
- jQuery (Autocompletado y validaciones)

## Estructura del proyecto:

- "index.html": Acceso al sistema (Login).
- "/html": Vistas principales (Menú, Depósito, Transacciones, Envío de dinero).
- "/assets": Recursos (CSS, JS, Imágenes).
